Frequently Asked Questions

REQUIREMENTS

What are the requirements?
  • Minecraft 1.7.10 or above.
  • Optifine (Latest version recommended).
I have a config file (shader options file). How do I install it?
  • Put the config file inside .minecraft\shaderpacks
  • Rename the config file to the shaderpack's name (extension included, ended with .txt)
    Example: TBB.zip uses the config TBB.zip.txt
  • Choose the shader.
What are the recommended video settings?
  • Graphics : Fancy
  • Render Distance : 8 or Above (I test at 8)
  • Smooth Lighting : Maximum
  • Smooth Lighting Level : 100%
  • Dynamic Lights : Fast / Fancy
  • Sky : On
Can I run this in Minecraft 1.X.X?

Shaderpacks are usually not affected by Minecraft version. The lowest version I recommend is 1.12.2 though.
Older than that, it's still useable but it may have some issues.



Should I change render quality and shadow quality in Shaders menu?
  • Render quality, you can lower it for sure. However, I'd recommend using the built in anti-aliasing (FXAA or TAA) instead of higher render quality since it is less taxing on FPS.
  • Shadow quality, don't. You can change it through shader options. Shadows > change Shadowmap Resolution and/or Shadow Distance.

VIDEO SETTINGS

My top half of the sky is broken.

Video Settings > Details > enable Sky.


I don't bring light sources and caves are too dark.

Video Settings > change Brightness.
This shader uses similar brightness to vanilla on caves (unless you change certain settings).
