What is Beyond Belief Shaders

Beyond Belief shaders project is focused around making a customizable and feature-rich Minecraft shader. Another major part of this project is trying to make an easily readable code to help out new shader developers. This is easily done by making every part of the shaders modular and by adding plenty of comments explaining what it does.

INSTALLATION

  • Download and install Optifine
  • Download the shaderpack
  • Put the shaderpack inside .minecraft\shaderpacks
  • Launch Minecraft
  • Open Options > Video Settings > Shaders
  • Choose the Shaders
